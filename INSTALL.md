# Installation

1. Recommended path:

		\Programs\Caps\
  
2. Place shortcut in the `Startup` directory:

 `Run:`  `shell:startup` 

 or,
 
		%AppData%\Microsoft\Windows\Start Menu\Programs\Startup\
  
3. Place shortcut to the `Keyboard-map` and `Readme` in Start menu:

		%AppData%\Microsoft\Windows\Start Menu\
